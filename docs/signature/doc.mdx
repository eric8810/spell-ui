## Installation

<InstallationTabs item="signature">
```vue file=../../vue/src/components/ui/Signature.vue
```
</InstallationTabs>

> Place the [LastoriaBoldRegular.otf](https://spell.sh/LastoriaBoldRegular.otf) font file inside your <code class="bg-muted/50 dark:text-neutral-300 text-neutral-600 p-1.5 rounded-md before:hidden after:hidden">public</code> directory or update the path in the component to load a different font.
