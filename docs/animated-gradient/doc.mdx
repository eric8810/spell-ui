## Usage

```vue
import { AnimatedGradient } from "@/components/ui";
```

```vue
<div class="relative h-[300px] w-full">
  <AnimatedGradient :config='{ preset: "Prism" }' />
  <div class="relative z-10">Your content here</div>
</div>
```

### Custom Configuration

```vue
<AnimatedGradient
  :config='{ preset: "custom",
    color1: "#1a1a2e",
    color2: "#16213e",
    color3: "#0f3460",
    rotation: 45,
    speed: 25,
    swirl: 60, }'
/>
```
